I'm not a professional chef, but I am an ✨aspiring web developer✨! This may not be a real catering business, but I do cook up cool websites like this one! This restaurant page was made with just basic HTML, CSS & javascript as an exercise for The Odin Project, a free online course that teaches web development.

The purpose of this project was to practice DOM manipulation by generating the pages in javascript rather than the standard way of just using static HTML. Ironically this "about us" page was the most complicated one to render using only javascript!
